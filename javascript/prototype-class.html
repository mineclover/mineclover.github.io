클래스의 개념

자바스크립트의 클래스는 다른 클래스와 조금 다르다
자바스크립트는 prototype 기반으로 동작한다

프로토타입을 이해하기 위해
new와 new로 만들어지는 인스턴스를 이해해야한다

prototype

왜 가져와지느냐

왜 fruits가 this로 지목되느냐
prototype에서 this는 호출된 메서드를 호출한 클래스가 들어있는 변수를 this로 잡는다

쉽게 말해서, 타입 내장 메서드

prototype은 선택한 클래스에 메서드를 선언할 수 있다

클래스가 상속된 변수들은 클래스에 선언된 메서드 들을 사용할 수 있다
this는 이러한 메서드를 부르는 상속된 변수를 참조한다

이러한 prototype 문법이 ES2015에서부터 class 문법 형태로 문법 변형한 것이다
