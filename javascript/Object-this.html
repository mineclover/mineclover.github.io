Call() 메서드

A.function.Call(B)
멤버로 선언된 일반 함수( or 객체 데이터) 를 B에게 빌려주는 역활을 하고
일반함수로 선언된 객체 데이터는 호출된 위치에서 정의되기 때문에
빌려졌을 때 B의 데이터를 사용하여 함수를 실행한다

B 에게 빌려준다
빌려줘서 function 을 호출한다

이러한 Call() 메서드의 대표적인 예시는 .forEach
