flex glow 로 비율 단위 버튼 넣을 때​

flex-glow , flex-shrink 비율 잡고자 할 수 있으나 내부 컨텐츠 사이즈 때문에 사이즈 조절이 어려울 수 있다 width 가 개별로 100% 로 있어도 사이즈가 배분되지만

내부 컨텐츠 사이즈로 인해 basis 가 생긴다 그래서 그냥 width % 사이즈로 최대사이즈 기준 비율을 계산해서 부여하는 것이 좋다 .. 예전에 안됬던 거 같은데 또 되네 .. * 에 box-size : border-box 를 붙여서 그런걸까

text 에 스타일 넣으려면 inline​

const InfoTextWrap = styled.div`

`;

  font-weight: 700;
`;

  setStep: React.Dispatch<SetStateAction<number>>;
