# 사용하는 문법.. 은 또 뭘까

constructor 내에
this.함수호출(객체) 를 전달하는 문법에 대해서…
그냥 객체를 통째로 던지는데
객체의 이름을 미리 리스트로 저장해두고
그 리스트를 forEach로 하나씩 가져오면서
객체[리스트] 형식으로 잘 꺼낸다

그리고 옵션에 미리 널을 대비한 옵션 설정 값들을 넣어두고
논리 연산자 || 를 활용해서 값이 없을 경우 미리 설정한 값을 실행하도록 하는 구조를 가지고 있음

Fonty 예제

Object.assign 대상 객체에 출처 객체의 속성을 복사해 넣는다

Object.assign(el.style, objectB)
약간 web-api로 있는 Object 기능이고
style.스타일속성 = 값; 넣으면 스타일이 적용된다
이러한 구조인 것을
Object.assign() 를 통해서 객체 데이터를 통째로 넣을 수 있다
